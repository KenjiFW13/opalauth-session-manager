# OpalAuth: Secure Session Manager 💎

Este projeto é o laboratório central da disciplina de Front End Engineering da FIAP. 
Aqui, aplicaremos 100% da nossa apostila de JavaScript de forma incremental.

## 🚀 Objetivo
Desenvolver um sistema de autenticação funcional que gerencia o estado da sessão 
do usuário utilizando persistência no navegador (LocalStorage/SessionStorage).

## 🛠️ Tecnologias e Metodologia
- **Linguagens:** HTML5, CSS3 e JavaScript (ES6+).
- **Versionamento:** GitFlow (Branches: master, develop, feature).
- **Gestão:** GitHub Projects (Kanban).
- **Mentoria:** Monitor Ariete (IA personalizada).

## 📋 Fluxo de Trabalho
1. Realize o **Fork** deste repositório.
2. Abra um **Draft Pull Request** apontando para o mestre.
3. Mova seu card no **Project** para a coluna "In Progress".
4. Siga as instruções de cada Issue enviada pelo Professor.
